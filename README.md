# Warships

Морской бой с возможностью сетевой игры. Делали командой из 3-х человек. Опубликован где либо не был.

В проекте занимался работой с профиль сервером, клауд скриптами, реализовывал регестрацию/логин игрока, лобби, комнаты, поиск игры. Использовались Playfab и Photon.

В плейфабе были созданы автоматизации для обработки событий создания игрока и ожидания верификации электронного адреса. Автоматизирована система отправки писем с верификационной ссылкой в момент регистрации и выдача базовых айтемов из внутреигрового магазина в плейфаб.
